# DataTest
Analisar dados de vendas de uma empresa fictícia para gerar insights de negócio, identificar padrões e apoiar decisões estratégicas.

Tarefas da planilha de vendas:

Caminho dos times até a final (quantos jogos, vitórias, empates, derrotas)
✔️ Distribuição de gols por fase
✔️ Desempenho do campeão Corinthians contra diferentes fases
✔️ Comparar campanhas de times grandes x pequenos
✔️ Artilharia por time e confronto (pode adicionar depois colunas de artilheiros)
